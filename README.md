# Zombie Jungle: Last Stand

A polished, dependency-free browser survival shooter. Hold a moonlit jungle clearing for 30 seconds against walkers, runners, droppers, and heavy infected.

## Play

Open `index.html` directly or serve the folder with any static web server.

## Controls

- Move: `A` / `D` or Left / Right arrows
- Jump: `W`, `K`, or Up arrow
- Aim upward: `E` or Down arrow
- Fire: `Space` or `J`
- Pause: `P`
- Mute: `M`
- Mobile: on-screen multi-touch controls

## Features

- Frame-rate-independent movement and combat
- Four enemy classes with escalating difficulty
- Combo scoring, accuracy report, and persistent high score
- Procedural layered sound effects and adaptive music
- Particles, camera shake, lighting, damage feedback, and animation
- Pause, volume controls, reduced-motion support, and responsive touch UI
- No build step or external runtime dependencies
